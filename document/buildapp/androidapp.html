Androidアプリにする

下のリンクから「ピコピコゲームAndroidアプリ化プロジェクト(PicoPicoGamesAndroid.zip)」をダウンロードします。

「ピコピコゲームAndroidアプリ化プロジェクト(PicoPicoGamesAndroid.zip)」

PicoPicoGamesAndroid.zipは解凍せず、そのまま利用します。

Eclipseを起動します。

環境設定から「Android」->「NDK」を開いて、

NDK Location に NDK へのパスを入力します。

右クリックして Import を選択。

「Exisiting Projects into Workspace」を選択。

「Select arcive file」を選択。

PicoPicoGamesAndroid.zipを選択。

Finishをクリック。

プロジェクトがインポートできました。

Assetを選択して、

右クリック

File System を選択して、

Browseボタンをクリック

DotEDITORから書き出したリソースがあるフォルダを選択。

ゲームリソースのフォルダ(ここでは CAR RACE ) に チェック入れて、

Finishをクリック。リソースをすべて読み込みます。これで準備ができました。ビルド＆実行するとピコピコゲームが動きます。

ビルドができない場合は、Java Compliter の Compiler Level が 1.6以上になっているか確認してください。

この後、作ったアプリを Google Play で公開するには、パッケージ名の変更やアイコンなどのリソースを用意する必要があります。詳しくは Androidアプリ関係の資料や Android デベロッパーページ等をご覧下さい。
