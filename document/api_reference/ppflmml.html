MMLを再生します。nは1から3までです。

ppflmml[n]:play(mml)

MMLを再生します。引数を省略した場合はpreload()したMMLを再生します。 MMLはFlMMLと互換性があります。

sample code

--画面をタッチでドレミファソラシドを再生
function start()
  if #pptouch() > 0 then
    ppflmml[1]:play(mml)
  end
end

mml="t120o5l4cdefgab<c"

ppflmml[n]:stop()

MMLの再生を止めます。

ppflmml[n]:pause()

MMLの再生を一時停止します。

ppflmml[n]:preload(mml)

MMLの事前読み込みを行います。

ppflmml[n]:resume()

一時停止したMMLの再生を再開します。

ppflmml[n]:isPlaying()

再生しているかどうかを調べます。

ppflmml[n]:isPaused()

一時停止しているかどうかを調べます。

ppflmml[n]:volume(vol)

ボリュームを指定します。引数が省略された場合は設定値を取得します。

ppflmml[n]:noteOn(track, noteNo, volume)

トラックを発音させます。noteNo は 0〜8オクターブx12の整数値です。例えば "o5c" の noteNo は 60 になります。

sample code

--画面をタッチで発音
ppflmml[1]:play("@3")
note=false

function start()
  if #pptouch() > 0 then
    if not note then
      ppflmml[1]:noteOn(1,67,1)
      note=true
    end
  else
    ppflmml[1]:noteOff(1)
    note=false
  end
end

ppflmml[n]:noteOff(track)

トラックの発音を停止します。
