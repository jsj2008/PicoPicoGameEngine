ppfont:loadTTF(ttffont, fontname,size)

TTFフォントを読み込みます。フォントはfontnameで識別します。
DotEDITORでは使用できません。

ppfont:loadTexture(texture, fontname,width,height)

指定したテクスチャをフォントとして使用します。フォントはfontnameで識別します。

ppfont:loadTexture(name, fontname,width,height)

フォントテクスチャ(png)を読み込みます。
DotEDITORでは使用できません。

ppfont:set(name)

使用するフォントを選択します。標準ではhalf、mini、defaultの３つが指定ができます。

sample code

--3つのフォントを縦に並べて表示します
function start()
  ppfont:set("default")
  ppgraph:print("HELLO")
  ppgraph:move(0,ppfont:height())
  ppfont:set("mini")
  ppgraph:print("HELLO")
  ppgraph:move(0,ppfont:height())
  ppfont:set("half")
  ppgraph:print("HELLO")
end

ppfont:height()

フォントの高さを取得します。

ppfont:width(string)

フォントの横幅を計算します。

ppfont:size(string)

フォントのサイズを計算します。

sample code

--画面の中央に"HELLO WORLD"を表示します。
function start()
  local s="HELLO WORLD"
  ppgraph:locate(ppgraph:layout(ppfont:size(s),true,true))
  ppgraph:print(s)
end

ppfont:texture()

選択中のフォントのテクスチャを返します。
