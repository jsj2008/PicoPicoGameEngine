CocosDenshionを使ったBGM再生を行います。 DotEDITORでは使用できません。

ppbgm.preload(filename)

BGMデータを先読みします。

ppbgm.play(filename)

BGMを再生します。

ppbgm.stop(close)

BGMを止めます。closeにtrueにするとBGMデータを破棄します。

ppbgm.pause()

BGMの再生を一時停止します。

ppbgm.resume()

BGMの再生を再開します。

ppbgm.rewind()

BGMを巻き戻します。

ppbgm.isPlaying()

BGM再生中かどうかを返します。

ppbgm.volume(vol)

ボリュームを設定します。引数がない場合は設定値を返します。
