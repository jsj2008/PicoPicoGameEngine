ボタンを処理するクラスです。

ppbutton(title)

titleを表示したボタンを作成します。

全選択

:draw()

ボタンを描画します。

:size()

ボタンのサイズを返します。

.width
.height

ボタンのサイズを指定します。ボタンのサイズはタイトル文字で自動設定されますが、このパラメータを変更することで任意のサイズに変更することができます。

:layout(auto,cx,cy,rx,ry,rw,rh)

ボタンのレイアウトを指定します。

sample code

全選択

:aabb()

ボタンのバウンディングボックスを計算します。

:drawbg(rect)

ボタンの背景画像を描画します。

:idle()
:idle({points})

ボタンのタッチ判定を処理します。

.bgcolor (default:ppgraph.black)

背景色です。nilにすると透明になります。

全選択

.selectcolor (default:ppgraph.red)

タッチ時のテキストカラーです。

.color (default:ppgraph.white)

非タッチ時のテキストカラーです。

.autolayout (default:false)
.centerx (default:false)
.centery (default:false)
.layoutarea (default:nil)

:layout()で使用するパラメータです。直接指定する事ができます。

.bg.tile (default:0)

背景画像をタイル番号で指定します。

.bg.texture (default:nil)
