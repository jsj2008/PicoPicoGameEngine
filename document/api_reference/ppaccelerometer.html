ppaccelerometer:start(interval)

加速度センサーを稼働させます。intervalでセンサーの計測間隔を指定できます。

sample code

ac=ppaccelerometer

ac:start(100)

function start()
  local v=ac:value()
  ppgraph:print("X:"..v.x)
  ppgraph:move(0,16)
  ppgraph:print("Y:"..v.y)
  ppgraph:move(0,16)
  ppgraph:print("Z:"..v.z)
end

ppaccelerometer:stop()

加速度センサーを止めます。

ppaccelerometer:isAvailable()

加速度センサーが利用可能か調べます。

ppaccelerometer:value()

加速度センサーの値を取得します。
